# CKTS-Utils

Utility scripts for TI-83 and TI-84 to automate functions for phasor analysis

## Uploading to Your Calculator

1. [Download TI Connect](https://education.ti.com/en/products/computer-software/ti-connect-ce-sw)
2. Open the files in the `scripts/` directory in Ti Connect while connected to your calculator
3. Click the following icon:
    - ![Send program to the connected calculator(s)](assets/upload.png)
4. Hit the "prgrm" button on your calculator and "CMPLXMAT" and "PHASRDEG" should be listed

## Inputting Phasor in Degrees

`phasor_deg` allows you to input the value for a phasor in degrees instead of having to convert to radians

**NOTE** - your calculator must be in radians mode when using this program

## Solving Complex Matrices

See: [Solving system of complex valued linear equations with TI83 or TI84](https://www.youtube.com/watch?v=inHqRSktULs&t=2s) for the exact process used in the script

Allows you to solve complex matrices. The result of the operation is stored in `L2` on the calculator
